# Oise au Jardin 🌿

A local gardening service website for the Oise region in France.

## 🌱 About the project
This website is designed to help people find gardening services such as:
- Lawn mowing
- Hedge trimming
- Garden maintenance

## 🚀 How to use
Just open the website in a browser.
